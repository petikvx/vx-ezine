Итак команды Windoze98...
~~~~~~~~~~~~~~~~~~~~~~~~~

   Некоторые  команды  просто вызывают api функций из разных библиотек, а
большенство просто виндовыми командами.

   Вызовы этих команд очень просты:

rundll32 {имя библиотеки api или команды}, {название самой команды}

winver
~~~~~~
   Это команда показывает информацию о винде...

rundll32 user,disableoemlayer
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Спровоцировать сбой.

rundll32 shell32,ShellAboutA
~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Info-Box (о Windows) вызвать, почти как команда winver.

rundll32 user,swapmousebutton
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
"Переставить" клавиши мыши *

rundll32 keyboard,disable
~~~~~~~~~~~~~~~~~~~~~~~~~
"Отключить" клавиатуру *

rundll32 mouse,disable
~~~~~~~~~~~~~~~~~~~~~~
"Отключить" мышь *

rundll32 user,setcursorpos
~~~~~~~~~~~~~~~~~~~~~~~~~~
Курсор мыши переместить в левый верхний угол

rundll32 shell32,SHExitWindowsEx 0
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Windows 98 перезагрузить (без autoexec.bat и др.)

rundll32 shell32,SHExitWindowsEx 1
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Работу с Windows 98 закончить

rundll32 shell32,SHExitWindowsEx 2
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Windows-98-PC boot

rundll32 shell32,SHExitWindowsEx -1
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Windows-98-Explorer перезагрузить

rundll32 user,exitwindows
~~~~~~~~~~~~~~~~~~~~~~~~~
Завершение работы Windows

rundll32 user,setdoubleclicktime 
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
установить новую скорость  двойного нажатия *

rundll32 krnl386.exe,exitkernel
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
выход из Винды без любых сообщений/вопросов

rundll32 shell,shellexecute
~~~~~~~~~~~~~~~~~~~~~~~~~~~
Explorer-Окно открыть в корневом каталоге
