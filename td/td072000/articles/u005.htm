  Описание параметров всех ключей для Windows File Protection или 
System File Checker (SFC):

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Winlogon 


SFCDisable (REG_DWORD)
~~~~~~~~~~~~~~~~~~~~~~

0 = это функция отключена (стандартный параметр в Win2k)
1 = включение этой функций, но потом будет включена снова(?)
2 = включение этой функций только после перезагрузки
4 = отключение функций

SFCScan (REG_DWORD)
~~~~~~~~~~~~~~~~~~~

0 = не сканировать защищенные файлы при загрузки
1 = сканировать защищенные файлы при каждой загрузке
2 = сканировать защищенные файлы всегда


SFCQuota (REG_DWORD)
~~~~~~~~~~~~~~~~~~~~

n = размер (в Мегабайтах) dllcache quota 
(стандартное 0xffffffff или приблизительно 300 MB) 
FFFFFFFF = кэш всех защищенных системных файлов на локальном жестком диске


SFCDllCacheDir (REG_EXPAND_SZ)
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
 
Path = локальная или сетевая позиция dllcache диры 
(стандартное %SystemRoot%\System32)


SFCShowProgress (REG_DWORD) 
~~~~~~~~~~~~~~~~~~~~~~~~~~~
    
0 = Прогресс Системы файла корректор не отображен (стандартное) 
1 = Прогресс Системы файла корректор отображен на мониторе.
