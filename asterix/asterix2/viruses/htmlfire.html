HTML.Fire is first in the world HTML/DOS hybrid. The virus infects all htm files in current directory. It uses VBScript in HTML (given version works under Internet Explorer 4.0 and above) to convert itself back to DOS executable by running debug with hex-dump script in shell.
In fact it is a direct action html-infector, that finds .htm files on local disc beeing runned, and places there a VBScript with hexdump to reborn this dropper again.
