Конструктор SBVM v0.01
(c) by Duke/SMF


   Пора уже приниматься за массовое производство BAT-вирусов. Этим-то,
собственно, и занимается конструктор SBVM - Simple Batch Virus Maker.
Версия 0.01 только пробная, но создает вполне работоспособные BAT-вирусы,
не детектирующиеся антивирусами. Для размножения вирусы используют
DOS-утилиту FIND и являются неработоспособными, если она отсутствует в
каталогах PATH.
   Конструктор - это файл SBVM.EXE, при запуске которого в качестве
параметра надо указать имя конфигурационного файла. Конфигурационный
файл надо предварительно отредактировать, указав следующие опции :

* A *
Имя автора вируса, которое будет указано в самом вирусе.

* B *
Имя файла, к который следует записать готовый вирус.

* C *
Метка-идентификатор вируса. По этой текстовой строке вирус будет проверять
файлы на повторное заражение и с помощью нее он будет размножаться
(используя DOS-команду FIND). В метке нельзя использовать символы
"%", "*", "?", "/", "\", "<", ">", "|", ".", ";", ":", ",".

* D *
Файлы, которые вирус будет удалять при запуске. Если оставить эту строку
конфигурационного файла пустой, то ничего удаляться не будет. Максимальное
число удаляемых файлов равно 25 (так вот захотелось, и все тут!).

   После создания CFG-файла надо запустить конструктор командой
       SBVM.EXE <CFG-файл>
   EnJoy !!!
