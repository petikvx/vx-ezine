++++++[ Tequila Bandita ]++++++

Name: Tequila Bandita MD5 Cracker by 3rror aka Neo2k8
Version: 1.3 

Info: Mit diesem MD5 Cracker, können wie der Name schon sagt MD5 Hashes gecracked werden. 
Er ist sehr schnell, da er auf eine Datenbank zugreift in welcher die entschlüsselten Hashes stehen. 
Ist der Hash dabei, so ist er in weniger als 3 Sek. geknackt. 
Die Datenbank kann ständig erweitert werden, hierfür muss lediglich eine Wordlist durch den Cracker 
eingelesen werden.Die Wörter weden dann automatisch in einen MD5 Hash umgewandelt und samt des 
Plain-Textes in der Datenbank abgelegt.
