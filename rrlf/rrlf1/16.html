OutlookXPUnlocker by Energy

This is a program to unlock files for Outlook XP.
The following file-extensions can be selected:
.exe, .com, .bat, .vbs, .js, .cmd, .reg, .msi, .scr, .url, .hlp, .inf
It is also possible to select a custom file-extension.
