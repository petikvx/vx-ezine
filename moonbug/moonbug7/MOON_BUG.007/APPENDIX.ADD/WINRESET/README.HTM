Exposition

"Why is Windows Restart now free?" you may ask yourself.

Well I came to realize that Windows Restart is a simple program; it is never going to make me a lot of money. I mean sure a little income from it was nice, but it was getting to be a pain to create registration packets. I also realized that what people need is an easy to use program so I stripped down Windows Restart and made it free. I mean who really needs a VBScript to restart Windows? So if you are need of such a program just contact me and I'll get it to you, but otherwise I'm through with the big bloated Windows Restart and am proud to present the new, improved Windows Restart.

Summary

Windows Restart allows users have quick access to a number of Windows actions and folders via user definable taskbar notification icons and shortcut keys. Windows Restart 98 provides users quick access to the following actions and items:

  • Shut Down Windows
  • Reboot Windows
  • Log Off Current User
  • Restart Windows
  • Exit to MS-DOS Mode
  • The Windows Explorer Shut Down Dialog
  • Minimize All Windows
  • Cascade Windows
  • Tile Windows Vertically
  • Tile Windows Horizontally
  • Undo Last Window Arrangement
  • Open Desktop Folder
  • Open Recycle Bin
  • Open Control Panel
  • Open Start Menu Folder
  • Open Recent Documents
  • Open Printers Folder
  • Open Network Neighborhood
  • Open Favorites Folder
  • Open My Computer Folder
  • Display Task List
  • Invoke Screen Saver
  • Open/Close CD-ROM Drive
  • The Windows Explorer Run Dialog

Installation

To install Windows Restart run the included Setup program. When installation completes an icon in the shape of a computer monitor will appear on the taskbar. To get started with Windows Restart, left-click or right-click on this icon. You can also access Windows Restart by clicking on the Windows Restart Menu icon in the Start Menu.

Changes Made to System by Installation

Windows Restart installs two executable files, two help files, and this HTML file to the directory you specify in the installation. Windows Restart also creates an entry in the HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Run registry key to run Windows Restart when Windows starts. Stratoware uninstallation will remove all of these changes in most cases.

Uninstallation

To remove Windows Restart:

  • Open the Control Panel.
  • Run Add/Remove Programs item.
  • In the list select Stratoware Windows Restart 98.
  • Click the Add/Remove button.
