VirusName:Redarc.623A
Aliases:Red Arc-623, RedArc.based, Redarc.623, Redar.623
Infection Length:623
Likelihood:rare
Target Platform:COM files


Description:

This virus only infects .COM files. Because this virus infects files, it can be caught through e-mail, BBSes, or the Internet. This virus has never been encountered by our customers. It is 623 bytes long. The Redarc.623A virus is a "direct action" virus. When an infected program has been launched, it immediately infects other programs. Like most viruses, the Redarc.623A virus does not contain a destructive payload. It does not employ encryption. This virus does not infect floppy or hard disk boot records. The Redarc.623A virus virus does not use "stealth" techniques. It infections cannot be repaired. Delete all infected programs and replace them with clean copies.


Comments:

No additional information.
